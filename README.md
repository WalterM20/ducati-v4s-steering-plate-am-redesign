# ducati-v4s-steering-plate-am-redesign
A comprehensive feasibility study redesigning the Ducati V4 S upper steering plate for LPBF Additive Manufacturing. This project leverages topology optimization (Altair Inspire) and FEM validation to compare AlSi10Mg and A20X aluminum alloys, achieving a 38% weight reduction and validating the economic viability of high-performance AM materials.
